saya ingin membuat sebuah bot telegram untuk menjual produk digital, produk digital yang ingin saya jual adalah GitHub Student Dev Pack. saya ingin flow pembeliannya adalah:

1. user mengunjungi bot tele saya.
2. user klik strart.
3. bot menampilkan list produk digital saya dan stoknya.
4. user klik produk dan bisa order lebih dari satu akun.
5. lalu user checkout dan bot mmebuatkan nomr order dan membuat tagihan, kemudian user melakukan pembayaran menggunakan API QRIS(coba cari penyedia pembayaran QRIS).
6. setelah pembayaran confirmed bot langsung mengirimkan detail produk berbentuk file text yang bisa user unduh.
